# Evolution Fantasy

Фэнтезийная MMOFPS игра в стиле аниме, вдохновленная "О моём перерождении в слизь" и "Ванпанчмен".

## О проекте

**Evolution Fantasy** - это многопользовательская онлайн-игра, где игрок начинает свой путь как простая слизь и эволюционирует в могущественного героя с огромным арсеналом способностей.

### Ключевые особенности:
- **Эволюционная система**: От слизи до сверхсущества
- **Мультиплеер**: PvP и кооперативные режимы
- **Аниме-стилистика**: Яркий, динамичный визуальный стиль
- **Глубокий геймплей**: Сложные комбо и стратегические способности

## Технологии

- **Unreal Engine 5.3+**
- **C++** для системной логики
- **Blueprints** для быстрого прототипирования
- **Multiplayer** с использованием Unreal Networking

## Быстрый старт

### Системные требования
- Windows 10/11
- Visual Studio 2022
- Unreal Engine 5.3+
- Минимум 16GB RAM

### Установка
1. Клонировать репозиторий
2. Запустить `Setup.bat` или `GenerateProjectFiles.bat`
3. Открыть `SoloGama.uproject` в Unreal Engine
4. Собрать проект

## Разработка

### Code Style
- C++: Google Style Guide с модификациями для UE
- Blueprints: Стандартизированные naming conventions
- Максимум 350 строк на класс

### Workflow
1. Создать ветку для фичи: `git checkout -b feature/название`
2. Реализовать изменения
3. Написать тесты
4. Создать Pull Request
5. Code Review и тестирование
