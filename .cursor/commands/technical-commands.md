# Technical Development Commands

## Code Generation
- `/implement-ability [name]` - Generate ability implementation code
- `/create-system [name] [type]` - Generate new game system framework
- `/implement-evolution [form]` - Generate evolution transformation code

## Unreal Engine Specific
- `/create-actor [name] [base_class]` - Generate UE Actor class
- `/create-component [name] [type]` - Generate UE Component
- `/create-blueprint [name] [template]` - Generate Blueprint template

## Testing & Validation
- `/test-system [name]` - Run automated tests for system
- `/validate-blueprint [path]` - Validate Blueprint for performance issues
- `/check-performance [system]` - Analyze system performance impact

## Build & Deployment
- `/build-development` - Build development version
- `/build-shipping` - Build optimized shipping version
- `/run-tests [suite]` - Execute test suite
