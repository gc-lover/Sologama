# Game Design Commands

## Ability Design
- `/design-ability [name] [type] [level]` - Design new character ability with stats and effects
- `/balance-ability [name] [context]` - Balance ability for PvE/PvP scenarios
- `/create-ability-tree [character]` - Create ability progression tree

## Character Design
- `/design-character [type] [evolution_stage]` - Design character concept and stats
- `/create-evolution-path [from] [to]` - Design character evolution progression
- `/balance-character [name] [level]` - Balance character stats and abilities

## Level Design
- `/design-level [name] [biome] [difficulty]` - Design level layout and challenges
- `/create-encounter [type] [difficulty]` - Design combat encounters
- `/balance-level [name]` - Balance level difficulty and rewards

## Quest Design
- `/design-quest [type] [difficulty]` - Design quest narrative and objectives
- `/create-quest-chain [theme]` - Design connected quest sequences
- `/balance-quest [name]` - Balance quest rewards and difficulty
