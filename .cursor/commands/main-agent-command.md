# Evolution Fantasy Development Commands

## Game Design Commands
- `/design-ability [name] [type] [level]` - Design new character ability
- `/design-level [name] [type] [complexity]` - Design game level
- `/design-character [type] [level]` - Design character concept
- `/balance-ability [name]` - Balance ability stats
- `/create-evolution-branch [from] [to]` - Create evolution path

## Implementation Commands
- `/implement-ability [name]` - Implement ability in code
- `/create-level [name]` - Create level in Unreal Engine
- `/implement-evolution [form]` - Implement character evolution
- `/generate-quest [type] [level]` - Generate quest content

## Testing Commands
- `/test-ability [name]` - Test implemented ability
- `/balance-check [system]` - Check system balance
- `/performance-test [feature]` - Test performance impact

## Utility Commands
- `/update-docs [section]` - Update project documentation
- `/create-spec [feature]` - Create technical specification
- `/validate-code [file]` - Validate code against standards
