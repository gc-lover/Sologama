# Основные команды Cursor для разработки игры "Evolution Fantasy"

## Команды для игрового дизайна

### Команда: Создать игровую механику
```
/create-game-mechanic [название] [описание]
```
Создает новую игровую механику эволюции персонажа с автоматической генерацией:
- Базовой реализации в C++
- Blueprint компонентов
- Документации в docs/game_design/mechanics/

### Команда: Разработать персонажа
```
/design-character [тип] [уровень эволюции]
```
Генерирует концепт персонажа с:
- Визуальным описанием в стиле аниме
- Статистикой способностей
- Эволюционным деревом
- Арт-описанием для художников

### Команда: Создать уровень
```
/create-level [тип] [сложность] [тема]
```
Автоматически генерирует:
- Дизайн уровня
- Расстановку врагов
- Систему прогресса
- Награды и loot

## Команды для технической разработки

### Команда: Создать игровую систему
```
/create-game-system [название] [тип]
```
Создает полную игровую систему:
- C++ классы
- Blueprint интеграцию
- Unit тесты
- Документацию

### Команда: Реализовать способность
```
/implement-ability [название] [тип] [уровень]
```
Генерирует:
- Логику способности
- Визуальные эффекты
- Баланс параметров
- UI элементы

### Команда: Создать UI компонент
```
/create-ui-component [название] [тип]
```
Создает UI компонент с:
- Unreal Slate кодом
- Стилизацией в аниме-стиле
- Локализацией
- Адаптивностью

## Команды для архитектуры

### Команда: Создать модуль
```
/create-module [название] [зависимости]
```
Генерирует:
- Plugin структуру
- Build файлы
- Интерфейсы
- Базовую реализацию

### Команда: Проанализировать производительность
```
/analyze-performance [система]
```
Проводит анализ:
- Профилирование кода
- Оптимизацию
- Рекомендации по улучшению

## Команды для контента

### Команда: Создать ассет
```
/create-asset [тип] [название] [параметры]
```
Генерирует спецификации для:
- 3D моделей
- Текстур
- Аудио
- Анимаций

### Команда: Импортировать ассет
```
/import-asset [путь] [тип]
```
Автоматически:
- Импортирует ассет в проект
- Настраивает параметры
- Создает превью
- Добавляет в систему версионирования

## Команды для тестирования

### Команда: Создать тест
```
/create-test [тип] [цель]
```
Генерирует:
- Unit тесты
- Integration тесты
- Performance тесты
- UI тесты

### Команда: Запустить тесты
```
/run-tests [фильтр]
```
Выполняет тестирование с:
- Автоматическими отчетами
- Скриншотами ошибок
- Логами выполнения

## Команды для документации

### Команда: Обновить документацию
```
/update-docs [раздел]
```
Автоматически обновляет:
- Техническую документацию
- Игровые механики
- API документацию
- Change log

### Команда: Создать гайд
```
/create-guide [тема] [аудитория]
```
Генерирует руководства для:
- Разработчиков
- Дизайнеров
- Тестировщиков
- Игроков
