# Специфические команды для игры "Evolution Fantasy"

## Команды эволюции персонажа

### Команда: Создать эволюционную ветку
```
/create-evolution-branch [от_формы] [к_форме] [условия]
```
Создает ветку эволюции с:
- Условиями трансформации
- Новыми способностями
- Визуальными изменениями
- Статистическими бонусами

### Команда: Балансировать способности
```
/balance-ability [способность] [уровень] [класс_врага]
```
Автоматически балансирует:
- Урон/защиту
- Стоимость ресурсов
- Кулдаун
- Эффективность против разных типов врагов

### Команда: Создать комбо-систему
```
/create-combo-system [способности] [последовательность]
```
Генерирует систему комбо с:
- Логикой последовательностей
- Визуальными эффектами
- Бонусами за успешное выполнение
- Штрафами за ошибки

## Команды мира игры

### Команда: Создать биом
```
/create-biome [тип] [уровень_сложности] [тема]
```
Создает игровой биом с:
- Генерацией ландшафта
- Системой погоды
- Фауной и флорой
- Уникальными механиками

### Команда: Сгенерировать квест
```
/generate-quest [тип] [уровень] [награда]
```
Автоматически создает:
- Сюжет квеста
- Цели и условия
- Диалоги NPC
- Систему наград

### Команда: Создать подземелье
```
/create-dungeon [тема] [размер] [босс]
```
Генерирует подземелье с:
- Процедурной генерацией
- Системой комнат
- Ловушками и механиками
- Финальным боссом

## Команды мультиплеера

### Команда: Создать кооперативный режим
```
/create-coop-mode [количество_игроков] [цель]
```
Реализует кооперативный режим с:
- Синхронизацией состояний
- Распределением ролей
- Системой возрождения
- Совместными наградами

### Команда: Реализовать PvP систему
```
/implement-pvp-system [тип] [правила]
```
Создает PvP систему с:
- Рейтинговой системой
- Балансом классов
- Анти-чит защитой
- Системой наказаний

### Команда: Создать гильдии
```
/create-guild-system [функции] [ограничения]
```
Реализует систему гильдий с:
- Созданием и управлением
- Иерархией участников
- Совместным контентом
- Экономикой гильдий

## Команды монетизации

### Команда: Создать систему доната
```
/create-donation-system [тип] [ограничения]
```
Реализует монетизацию с:
- Балансом косметики
- Системой подписок
- Бонусами за поддержку
- Анти-пей2вин механиками

### Команда: Балансировать экономику
```
/balance-economy [ресурс] [источники] [стоки]
```
Анализирует и балансирует:
- Цены на предметы
- Дроп-рейты
- Стоимость крафта
- Экономику гильдий

## Команды аналитики

### Команда: Создать метрики
```
/create-metrics [событие] [параметры]
```
Добавляет аналитику для отслеживания:
- Игрового прогресса
- Поведения игроков
- Эффективности контента
- Технических показателей

### Команда: Создать A/B тест
```
/create-ab-test [функция] [варианты] [метрики]
```
Настраивает A/B тестирование с:
- Разделением аудитории
- Сбором данных
- Автоматическим анализом
- Рекомендациями по внедрению

## Команды поддержки

### Команда: Создать систему отчетов
```
/create-report-system [тип_проблемы] [приоритет]
```
Реализует систему отчетов с:
- Категоризацией багов
- Приоритизацией
- Автоматическим тикетингом
- Системой обратной связи

### Команда: Создать туториал
```
/create-tutorial [тема] [сложность] [формат]
```
Генерирует обучающий контент:
- Интерактивные гайды
- Видео-уроки
- Подсказки в игре
- Систему достижений
